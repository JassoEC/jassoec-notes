# Pensar en objetos

## Tener una historia de donde partir
  Historias de usuario: narrar como ocurre el problema o requerimiento que se trata de resolver

  **Elementos**
  
  - Titulo
  - Cómo
  - Quiero hacer...
  - Para que lo quiero hacer...

  ## Comenzar a entender los objetos
  *Existen dos tipos*
  - **Físcos** (usuarios, productos, pagos)
  - **Conceptuales** (existen en el concepto de lo que se esta resolviendo, por ejemplo: cuentas de usuario, carrito de compras, sesiones)
